This is my first website made with python flask

How to set it up:
  1. Create a virtual enviroment for python.
  2. Run this command in a terminal: python -m venv venv
  3. Activate the virtual enviroment
  4. In Windows run the command:  .\venv\Scripts\activate
  5. In Mac and Linux run the command: source venv/bin/activate
  6. When in the virtual enviroment run the commmand: pip install -r requirements.txt
  7. To run the program in your virtual enviroment run the command: python app.py
  8. Now you can go onto a web browser and type in: localhost:5000
  9. To deactivate the virtual enviroment just enter: deactivate
